# Fork

Best way to propose modifications to someone else’s repo

1. Fork the repository

- Go to the repo on GitHub
- Click “Fork”
- This creates a copy under your account, where you have full control.

2. Clone your fork to your local machine

```sh
git clone https://github.com/YOURUSERNAME/REPO.git
cd REPO
```

3. Create a new branch for your changes

This keeps your fork’s main branch clean.

`git checkout -b feature-my-improvement`

4. Make your changes

Edit files, commit:

```sh
git add .
git commit -m "Explain clearly what you changed"
```

5. Push the branch to your fork

`git push origin feature-my-improvement`

6. Create a Pull Request (PR)

Go to your fork on GitHub → You will see a banner:

- “Compare & pull request”
- Click that.
- Write a clear description of the change
- Reference any issues 
- Submit the PR

The repository maintainers will review your PR and decide to merge, request changes, or discuss.
